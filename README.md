# Yelena Inventory

Inventory management and stock counting system built for **Pamit Store**.

## Repository Structure

```
YelenaInventory/
│
├── YelenaInventory.Server/    # ASP.NET Core Web API
├── yelena_inventory/          # Flutter application
├── assets/                    # Logos, branding, icons and mockups
├── docs/                      # Project documentation
└── scripts/                   # Utility scripts
```

## Technology Stack

### Backend

* ASP.NET Core 9
* Entity Framework Core
* PostgreSQL

### Mobile

* Flutter
* Riverpod
* Drift (SQLite)
* Hive

## Current Features

* Branch management API
* Employee selection
* Local database
* REST API integration
* Offline-ready architecture

🚧 Under active development.
